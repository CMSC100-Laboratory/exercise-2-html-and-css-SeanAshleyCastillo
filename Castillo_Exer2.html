Project

Big Iron is a two-player real-time game that is skill-based, where the two players will try to shoot each other. There are several power-ups available in the game which will spawn while the round is ongoing. Additionally, there is a train that blocks bullets from the player and it moves back and forth from the edges of the screen. It is a first to five game where every time a player gets shot the other player gets a score.
